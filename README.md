# mapbox

ビルトインサーバーが一番楽

カレントディレクトリで下記コマンドを実行<br>
```
php -S localhost:8000
```

ブラウザから以下URLでアクセスすればOK<br>
http://localhost:8000

- api.php JSONの吐き出し
- mapbox.js 必要なJavaScriptファイル
- index.php エントリーポイント
- json/feature.json JSONレコードを格納
